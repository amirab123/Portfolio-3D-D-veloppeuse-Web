# 🚀 Amira Portfolio 3D

Bienvenue dans mon portfolio personnel, développé avec HTML, CSS et Tailwind CSS.

Ce projet présente mes compétences en développement web à travers une interface moderne, interactive et inspirée d’un style **cartoon + 3D**.

---

## 🎨 Aperçu

Portfolio moderne avec :
- Effets 3D interactifs
- Design coloré et animé
- Interface responsive
- Formulaire de contact (EmailJS)

---

## ⚙️ Technologies utilisées

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- EmailJS (contact Gmail)

---

## ✨ Fonctionnalités

- 🎨 Design cartoon moderne
- 🧊 Effets 3D sur les cartes
- 📱 Responsive (mobile / desktop)
- 📩 Formulaire de contact fonctionnel
- ⚡ Interface rapide et légère

---

## 🎯 Objectif

Ce projet a été créé pour :
- Présenter mes compétences en frontend
- Mettre en valeur mes projets
- Démontrer un style UI/UX créatif
